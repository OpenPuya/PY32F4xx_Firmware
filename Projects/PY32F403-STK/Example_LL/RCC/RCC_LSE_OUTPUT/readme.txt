                             Sample Description

功能描述：
此样例演示了使能LSE，并通过MCO引脚输出LSE时钟。

Function descriptions:
This example demonstrates enabling LSE and outputting LSE clock through MCO pin.
================================================================================
测试环境：
测试用板：PY32F403_STK
MDK版本： 5.28
IAR版本： 9.20
GCC 版本：GNU Arm Embedded Toolchain 10.3-2021.10

Test environment:
Test board: PY32F403_STK
MDK Version: 5.28
IAR Version: 9.20
GCC Version: GNU Arm Embedded Toolchain 10.3-2021.10
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 可由示波器观察到PA08输出LSE波形。

Example execution steps:
1. Compile and download the program to the MCU.
2. Observe the LSE waveform on PA08 using an oscilloscope.
